Usage The offcanvas plugin utilizes a few classes and attributes to handle the heavy lifting: .offcanvas hides the content .offcanvas.show shows the content .offcanvas-start hides the offcanvas on the left .offcanvas-end hides the offcanvas on the right .offcanvas-bottom hides the offcanvas on the bottom Add a dismiss button with the data-bs-dismiss="offcanvas" attribute, which triggers the JavaScript functionality. Be sure to use the button element with it for proper behavior across all devices.
